# digiself-realtime-api-sample

A collection of sample applications demonstrating the use of Digiself Realtime API.

## Samples

| Sample | Description |
|--------|-------------|
| [livekit](./livekit/) | Real-time avatar interaction using LiveKit |
| [meeting-bot](./meeting-bot/) | Avatar interaction in Google Meet using meeting bot |

Please refer to the README in each directory for detailed usage instructions.
